# 🛒 Mini Grocery Delivery App

A simple Android grocery delivery application built using **Kotlin** and **XML layouts**.  
This project was developed as part of an Android Developer Internship assignment and also marks my **first experience working with XML-based Android UI development**.

---

# 🚀 Features

- 📱 Login with Phone Number
- 🔐 OTP Verification (Demo OTP: 1234)
- 🛍️ Grocery Product Listing
- 🛒 Add to Cart Functionality
- ➕ Increase / Decrease Quantity
- 💳 Checkout Flow
- ✅ Order Success Screen
- 🎨 Clean and Responsive UI

---

# 🛠️ Tech Stack

- Kotlin
- XML Layouts
- RecyclerView
- Android Studio
- Intent Navigation
- MVVM (Basic)

---

# 📂 App Flow

1. Login Screen  
2. OTP Verification  
3. Home/Product Screen  
4. Cart Screen  
5. Checkout Screen  
6. Order Success Screen  

---

# ▶️ How to Run

1. Clone this repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Run the app on emulator or physical device
